## Dépendances du projet RC Car Simulator (C++)

# SFML (version >= 2.6.0 recommandée) # Déjà incluse dans le dépôt sous /SFML-2.6.2
Pour installation système :
    Mac : brew install sfml
    Linux : sudo apt install libsfml-dev

# Bibliothèque JSON (optionnelle pour une eventuelle sauvegarde de la configuration)
    # Mac : brew install nlohmann-json
    # Linux : sudo apt install nlohmann-json3-dev

# Boost (optionnel pour solveurs différentiels)
    # Mac : brew install boost
    # Linux : sudo apt install libboost-all-dev
